# Microcosmic Expansion Equation (MEE)

## 核心方程（纯文本版，便于复制）

U_{n+1} = U_n ∥ (U_n ⊕ F(x) + ∞-branch + η_min)

## 演化过程

1. 初始版本  
   U_{n+1} = U_n ∥ U_n ⊕ F(x)

2. 最终版本（加入生存机制）  
   U_{n+1} = U_n ∥ (U_n ⊕ F(x) + ∞-branch + η_min)

## 符号说明

- ∥ : 并行扩张（parallel expansion）
- ⊕ : 状态融合（fusion）
- ∞-branch : 无穷分支项（infinite branching）
- η_min : 最小扰动（minimal noise injection）
  - 防止模式崩溃（mode collapse）
  - 防止模型自噬（model autophagy）
  - 任何生成系统必须保留的生存不确定性底线

## UnI 模块（已整合）

UnI = ∞-branch + η_min
